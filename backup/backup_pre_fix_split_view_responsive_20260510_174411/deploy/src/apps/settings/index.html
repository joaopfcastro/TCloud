Configurações

App de configurações do sistema

Painéis
Tudo aqui roda no mesmo domínio do TCloud e usa o token enviado pela shell via postMessage.

Visão geral

Resumo do sistema, do armazenamento e das preferências da shell. Campos com Requer reinício só entram em vigor após reiniciar o serviço.

Servidor

Configurações definidas pelo schema do backend. Segredos continuam mascarados e só são revelados quando você usar o olhinho do campo.

Armazenamento

Resumo do cache em disco, dos itens em No Servidor e dos arquivos em Temporário (24h).

Compartilhamento

Políticas de links públicos, inventário de itens publicados e controles de acesso externo.

Interface

Preferências locais deste navegador para a shell Web, sem alterar o runtime do servidor.

Aplicativos

Instale apps por ZIP ou GitHub, governe permissões, habilite ou desabilite apps e acompanhe a trilha de auditoria do runtime.

Diagnóstico

Estado do runtime, conectividade principal, chaves pendentes de reinício e apps instalados.

Detalhes do item

Informações completas do arquivo selecionado.
